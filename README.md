# ระบบแสดงความคิดเห็นเพื่อทบทวนแผนบูรณาการฯ ช่วงเทศกาล และช่วงวันหยุด พ.ศ. 2570

ระบบรับฟังความคิดเห็นรายหัวข้อ/รายมาตรการต่อร่างแผนบูรณาการป้องกันและลดอุบัติเหตุทางถนน
ช่วงเทศกาลและช่วงวันหยุด พ.ศ. 2570 (ใช้แทน Google Form) — Node.js + Express, เก็บข้อมูลเป็น JSON

- ฟอร์มกรอกความเห็น: `/`
- หน้าผู้ดูแลระบบ: `/admin` (ต้องใส่ ADMIN_TOKEN)

## การรันในเครื่อง
```
npm install
ADMIN_TOKEN=xxxx PORT=3200 node server.js
```

## Deploy บน Railway
ตั้งค่าใน Railway:

| สิ่งที่ต้องทำ | ค่า |
|---|---|
| **Variables** → `ADMIN_TOKEN` | รหัสยาว ๆ เดายาก (ใช้เข้าหน้า /admin) |
| **Variables** → `DATA_DIR` | `/data` |
| **Volume** → Mount path | `/data` (เก็บคำตอบถาวร ไม่หายตอน redeploy) |

> `PORT` ไม่ต้องตั้ง Railway กำหนดให้เอง (โค้ดอ่าน `process.env.PORT`)

ครั้งแรกที่ deploy ระบบจะคัดลอกข้อมูลตั้งต้น (`plan.json`, `agencies.json`) เข้า volume อัตโนมัติ
ส่วนคำตอบของผู้กรอก (`responses.json`) ไม่ถูกเก็บใน git (อยู่ใน `.gitignore`) และจะอยู่บน volume เท่านั้น

## โครงสร้างข้อมูล
- `data/plan.json` — ร่างแผน 11 หัวข้อ (ปีใหม่/สงกรานต์ และวันหยุด แยกชุดมาตรการ/ตัวชี้วัด/นิยาม)
- `data/agencies.json` — หน่วยงานส่วนกลาง + รายชื่อจังหวัด + ประเภทหน่วยงานจังหวัด
- `data/responses.json` — คำตอบที่ผู้กรอกส่งเข้ามา (สร้างอัตโนมัติ ไม่ขึ้น git)
